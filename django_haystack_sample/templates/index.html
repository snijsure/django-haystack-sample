Searchly Sample Django Application.


This example illustrates basic search features of SearchBox.io (ElasticSearch as service).

Each CRUD operation on documents is reflected to search index in real time.

To test SearchBox.io's search features login to admin screen and create new documents.

Sample application is using Haystack moduler search for Django and pyelasticsearch (toastdriven fork) Python ElasticSearch client to integrate with SearchBox.io.
